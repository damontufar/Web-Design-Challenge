Summary: Latitude vs. X

The purpose of this project was to analyze how weather's cities are affected by their closeness with the equator. The methodology followed included te extraction of random coordinates' from the OpenWeatherMap API in order to analyze certain data of 500 cities.

Once having the results, Matplotlib library was used to plot differents measures with the purpose of finding insights from the relationships between weather and latitude, including: temperature, humidity, cloudiness and Wind speed.
